# Resumen
Este proyecto es una aplicación de escritorio desarrollada en Java utilizando JavaFX. El objetivo principal de este ejercicio es aprender a realizar validaciones de formato de texto mediante expresiones regulares (RegEx), controlando además la alineación visual de los componentes en la interfaz gráfica.

## Características
Validación con Expresiones Regulares: Evalúa mediante patrones matemáticos de texto (matches) que la entrada contenga única y exclusivamente caracteres numéricos.

Restricción de Longitud Fija: Aplica una regla estricta para garantizar que el número introducido tenga exactamente 10 dígitos.

Alineación Centrada Automatizada: Utiliza la propiedad Pos.CENTER en el contenedor VBox para forzar a que todos los controles visuales permanezcan perfectamente agrupados en el centro geométrico de la pantalla.

Feedback Dinámico Bicolor Implícito: Cambia el contenido del control Label en tiempo real informando al usuario si el formato introducido es correcto o no.
